## Features

- 6-letter Hexle-style gameplay with 6 attempts
- 22,000+ word dictionary
- Color-coded feedback (🟩 correct, 🟨 present, ⬜ absent)
- On-screen responsive keyboard (mobile friendly)
- "Not in word list" toast notifications
- Game over reveals the answer

## Build & Run

### Docker Compose (recommended)
```sh
docker compose up --build
```
- UI → http://localhost:3000
- API → http://localhost:8080

### API only
```sh
./gradlew :api:bootRun
```

### UI only
```sh
cd ui && npm install && npm run dev
```

## Kubernetes Deployment

```sh
# Build & push ARM64 images to Docker Hub
./build-and-push.sh

# Deploy to cluster
kubectl apply -f k8s/
```
